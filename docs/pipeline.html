Offline Python Pipeline

Five scripts that run once to build the vector index. Hover over participants for details.

sequenceDiagram
  participant S1 as 01_download
  participant S2 as 02_export_onnx
  participant S3 as 03_embed
  participant Triton as Triton gRPC
  participant S4 as 04_ingest
  participant Qdrant as Qdrant REST
  participant S5 as 05_enrich

  S1->>S1: Download CMU corpus (42,306 plots)
  S2->>S2: Export all-MiniLM-L6-v2 → ONNX
  S3->>Triton: Batch text chunks (gRPC :8001)
  Triton-->>S3: float32[384] embeddings
  S4->>Qdrant: Upsert points (REST :6333)
  Qdrant-->>S4: OK
  S5->>TMDB: Fetch poster URLs (40 req/s)
  TMDB-->>S5: thumbnail_url per movie
  S5->>Qdrant: Patch payload thumbnail_url
