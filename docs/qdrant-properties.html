QdrantProperties

Spring Boot @ConfigurationProperties class that binds the qdrant.* namespace from application.yml, exposing the Qdrant base URL and mock flag to the rest of the application.

Configuration Properties

Property (YAML key)Java fieldTypeDefaultDescription
qdrant.base-url baseUrl String http://localhost:6333 Root URL of the Qdrant REST API. Used by VectorSearchServiceImpl to build the search endpoint URL.
qdrant.mock mock boolean false When true, Spring wires MockVectorSearchService instead of VectorSearchServiceImpl. Allows full search-flow testing without a running Qdrant container.

application.yml defaults

qdrant: base-url: http://localhost:6333 mock: false

Class Definition

@ConfigurationProperties(prefix = "qdrant") public class QdrantProperties { private String baseUrl = "http://localhost:6333"; private boolean mock = false; public String getBaseUrl() { return baseUrl; } public void setBaseUrl(String baseUrl) { this.baseUrl = baseUrl; } public boolean isMock() { return mock; } public void setMock(boolean mock) { this.mock = mock; } }

Mock vs Real Wiring

The qdrant.mock flag drives a @ConditionalOnProperty split between the two VectorSearchService implementations:

qdrant.mock valueBean wiredBehavior
true MockVectorSearchService Returns fixed mock results without contacting Qdrant. Default for unit tests.
false (or absent) VectorSearchServiceImpl Calls the real Qdrant REST endpoint at baseUrl.
