IngestIntoQdrantServiceImpl

Implements IngestIntoQdrantService. Runs pipeline/04_ingest_qdrant.py via PythonScriptRunner. Skips execution when the Qdrant movies collection already has points and force=false.

Idempotency Check

When force=false, the service calls GET <qdrant.baseUrl>/collections/movies and inspects the JSON response field result.points_count. If points_count > 0, the collection is already populated and the script is skipped. On any HTTP or parse error, the check returns false and the script runs.

Execute Flow

flowchart TD
  A[execute(onLog, force)] --> B{force=false?}
  B -- No --> D
  B -- Yes --> C[GET /collections/movies\npoints_count > 0?]
  C -- Yes --> Skip[onLog: "already ingested, skipping."\nreturn]
  C -- No / error --> D[pythonScriptRunner.run(pipeline/04_ingest_qdrant.py, onLog)]
  D --> E{exit 0?}
  E -- Yes --> F[return]
  E -- No --> G[throw IngestIntoQdrantServiceException]
