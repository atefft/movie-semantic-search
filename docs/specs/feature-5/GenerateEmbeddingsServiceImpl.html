GenerateEmbeddingsServiceImpl

Implements GenerateEmbeddingsService. Runs pipeline/03_embed_corpus.py via PythonScriptRunner to batch-embed 42k movie plot summaries. Skips execution when output files already exist and force=false.

Idempotency Check

When force=false, the service checks for both output files under project.root. If both exist, it logs a skip message and returns without running the script.

Output filePath (relative to project.root)
embeddings.npy data/embeddings/embeddings.npy
metadata.json data/embeddings/metadata.json

Execute Flow

flowchart TD
  A[execute(onLog, force)] --> B{force=false AND\nboth files exist?}
  B -- Yes --> C[onLog: "Embeddings already generated, skipping."\nreturn]
  B -- No --> D[pythonScriptRunner.run(pipeline/03_embed_corpus.py, onLog)]
  D --> E{exit 0?}
  E -- Yes --> F[return]
  E -- No --> G[throw GenerateEmbeddingsServiceException]
