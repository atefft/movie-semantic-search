ProjectProperties

Spring Boot configuration properties bound to the project.* prefix. Supplies project.root — the working directory for all docker compose and Python script invocations.

Configuration Properties

PropertyTypeRequiredDescription
project.root Path Yes Absolute path to the repository root. Used as the working directory for docker compose commands and Python script execution.

Class Declaration

@Validated @ConfigurationProperties(prefix = "project") public class ProjectProperties { @NotNull private Path root; public Path getRoot() { return root; } public void setRoot(Path root) { this.root = root; } }

application.yml Example

project: root: /path/to/movie-semantic-search

Usage

Injected into DockerComposeRunnerImpl, GenerateEmbeddingsServiceImpl, IngestIntoQdrantServiceImpl, and EnrichWithTmdbServiceImpl to resolve script paths and set the working directory. The @NotNull constraint ensures the application fails fast at startup if the property is missing.
