EnrichWithTmdbServiceImpl

Implements EnrichWithTmdbService. Runs pipeline/05_enrich_tmdb.py with TMDB_API_KEY injected as an environment variable. Skips execution when at least one Qdrant point already has a non-null thumbnail_url and force=false.

Idempotency Check

When force=false, the service calls POST <qdrant.baseUrl>/collections/movies/points/scroll with a filter that selects points where thumbnail_url is not null, limit 1. If the response contains at least one point, enrichment is considered done and the script is skipped.

Scroll Request

POST /collections/movies/points/scroll Content-Type: application/json { "filter": { "must": [{ "is_not_null": { "key": "thumbnail_url" } }] }, "limit": 1 }

Execute Flow

flowchart TD
  A[execute(onLog, force)] --> B{force=false?}
  B -- No --> D
  B -- Yes --> C[POST /points/scroll\nwith thumbnail_url filter\npoints.size > 0?]
  C -- Yes --> Skip[onLog: "TMDB enrichment already done, skipping."\nreturn]
  C -- No / error --> D[pythonScriptRunner.run(pipeline/05_enrich_tmdb.py, onLog,\nenv: TMDB_API_KEY=...)]
  D --> E{exit 0?}
  E -- Yes --> F[return]
  E -- No --> G[throw EnrichWithTmdbServiceException]

Script

PropertyValue

Idempotency endpoint POST <qdrant.baseUrl>/collections/movies/points/scroll
